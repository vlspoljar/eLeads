# eLeads
Java EE REST API

Tech stack:
- Java 8
- Java EE
- Hibernate 5
- RESTful Web Services
- MySQL
- WildFly 18

At location src/main/resources/db/database.sql you can find SQL script for database creation as well as tables creations and corresponding inserts.

Al location src/main/resources/json/eLeads.postman_collection.json you can find Postman Collection JSON file with examples and descriptions for all 15 endpoints in project.

War file can be found at location target/eLeads.war and it can be deployed at application server (WildFly).
